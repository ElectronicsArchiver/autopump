==========

## Included
 - VueJs + Vite
 - Express server with routes and Socket.io + Dynamic Crons
 - MongoDB
 - Docker + Nginx for deploy to heroku
 - PlatformIO + ESP8266

## Run dev
Copy ```.env.example``` from both client and server folder to ```.env``` file.
### Client
1) ```cd client```
2) ```yarn install```
2) ```yarn dev```
### Server
1) ```cd server```
2) ```yarn install```
2) ```yarn dev```
